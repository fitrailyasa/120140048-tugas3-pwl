# Tugas Individu 3 PWL (Pemrograman Web Lanjut)

### Clone Repo

```
git clone https://github.com/fitrailyasa/120140048-tugas3-pwl.git
```

### Pindah Direktori

```
cd 120140048-tugas3-pwl
```

### Instalasi Dependensi

```
npm install
```

### Perintah Mulai

```
npm start
```

### Perintah Testing

